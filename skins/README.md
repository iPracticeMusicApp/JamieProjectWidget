

## Decorative Border Skins

The following skins add stronger outer borders and decorative edge treatments:

- `cathedral-gold.css`
- `stained-glass.css`
- `royal-purple.css`
- `art-deco-navy.css`
- `rose-gold.css`
- `celtic-green.css`

These files can be selected by the administrative Widget Skin section after
they are uploaded to the repository's `skins/` folder and **Refresh Skins** is
clicked.
