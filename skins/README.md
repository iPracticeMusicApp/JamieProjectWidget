# Available Widget Skins

Included:

- `classic-blue.css`
- `midnight-dark.css`
- `church-burgundy.css`
- `forest-green.css`
- `modern-white.css`

To create another skin:

1. Copy `classic-blue.css`.
2. Rename it using lowercase letters, numbers, hyphens, or underscores.
3. Change the CSS variables and optional selector overrides.
4. Use the filename without `.css` as `DEFAULT_WIDGET_SKIN` in `index.html`.
